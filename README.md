## 🟠 Hello

I design and ship production systems that have to stay up, stay correct, and stay fast under real load. Over 15+ years I've worked across gaming, housing, fraud detection, and workforce platforms, usually somewhere between the database and the deploy pipeline. I care about calm software, clear writing, and tools that respect the people using them.

Lately I'm leaning hard into AI-assisted engineering: how teams keep trust, review, and documentation intact when a lot of the code is written with an agent in the loop.
